Stock Market Analysis

I created this these algorithms and analysis tools to practice my software/ML skills and learn more about the market, all while potentially making some money on the side. There's a few tools that are currently availabe for use:

Financial Technical Indicator Analysis

Use existing technical indicators (MACD, RSI, OBV) as inputs into machine learning (SVR) to predict percent price changes. Parameters for data preprocessing and SVR algorithm are user adjustable.

Twitter Sentiment Analysis

Use NLP to correlate tweets with market changes and use Naive Bayes to predict stock performance from recent tweets. Parameters for data preprocessing and NB algorithm are user adjustable.

Merger Arbitrage Analysis

Find the expected value of acquisition for a publicly traded company with an announced acquisition using a market reference. At each user specified percent change in probabiliy of acquisition, provide the top tweet with user specified search text for that date of change.

There is not a lot of form validation, so just double check your inputs if you hit errors. You should definitely put all your life savings in this.
