Stock Analysis

Hello there, and welcome to my lazily formatted web app!

The goal of this web app is to use machine learning and algorithms (cue other fancy terms) to predict market prices and make you lots of money (or lose lots of money - who really knows, just flip a coin).

This web app currently supports one lonely method of analysis:
1. Technical Indicator Analysis

Using technical indicators, I create pretty plots and use the outputs of technical indicators as inputs into a machine learning algorithm (SVR) to predict price changes. A lot of parameters are user adjustable so have it. If you don't know what they mean, too bad look them up on Google.

Once upon a time, I had a cool Twitter sentiment analyzer and predictor for market prices, but Twitter decided to lock the secret doors that GetOldTweets3 (Tweet scraping library) used. Lame.

With the rise of the reddit mob (hi GameStop 2021), I'd like to use a reddit API to analyze reddit sentiment concerning stock. When I get some free time... Eventually.

LASTLY, I was too lazy to put in form validation. So if you get errors, it's your own dang fault. Definitely not my code ;)
