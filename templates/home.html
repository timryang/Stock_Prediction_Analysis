Stock Analysis

Background

Welcome to my stock analysis web app!
My goal is to use machine learning theory and algorithms to provide market indicators and reduce risk for investment decisions.

This web app currently supports two methods of analysis:
1. Twitter Sentiment
2. Mutliptle Stock Dependencies

Twitter Sentiment uses the Naive Bayes algorithm. However, it currently does not support collecting more than ~500 tweets due to a timeout limitation on Heroku's server. Background processing has not been implemented.

Multiple Stock Dependencies currently uses the logistic regression, SVM, and KNN algorithms.

Finally, there is no form validation or error catching. So if you get an error, it might be likely that your inputs are bad or you exceeded the timeout limitation (which again should be around ~500 tweets).

Select a link at the top of the page to start! - Tim
